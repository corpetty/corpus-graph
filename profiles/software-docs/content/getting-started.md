# Getting started

Quill is a server-first web framework. **A new Quill app renders on the server by default and needs no client build step to return HTML.**

## Install

```
npm create quill@latest
```

## Your first route

The [[Router]] turns a file under `routes/` into a handler. The [[Template engine]] renders the response.

See the [[Routing guide]] for the full model.
